|------|------|

## 특수 기능

| 기능 | 설명 |
|------|------|
| 퇴직금 계산 | 1일 평균임금 × 30일 × (재직일수/365) |
| 평균임금 산정 | 3개월 총임금 ÷ 실제 역일수 (91일 고정 아님) |
| 통상임금 산정 | 월 통상임금 ÷ 209 × 8 (소정근로시간 기준) |
| 연차수당 계산 | 미사용 연차 × 1일 통상임금 |
| 연차 자동계산 | 근로기준법 기준 (1년 미만: 월 1일, 1년+: 15일, 2년마다 +1일, 최대 25일) |
| 최적 퇴사일 추천 | 30일 범위 내 퇴직금 최대화 날짜 탐색 |
| 등급 판정 | 퇴직금 기준 S~F 5등급 매핑 |
| PDF 리포트 | html2canvas → jsPDF 멀티 페이지 한글 PDF |
| 이미지 캡처 | html2canvas 결과 영역 PNG 저장 |
| PWA | 홈 화면 추가, 오프라인 지원 |
| View Transitions | 입력→결과 화면 전환 애니메이션 |
| Confetti | 등급 공개 시 3방향 컨페티 |
| Lottie | 등급 카드 축하 애니메이션 (인라인 JSON) |

## 계산 알고리즘 (근로기준법 기준)

```
퇴직금 = 1일 평균임금 × 30 × (재직일수 / 365)

1일 평균임금 = 3개월 총임금 / 실제 역일수
  - 3개월 총임금 = (월급 + 월 상여금) × 3
  - 실제 역일수 = 퇴사일 기준 직전 3개월의 달력 일수 (91일 아님)

1일 통상임금 = 월 통상임금 / 209 × 8
  - 209 = 월 소정근로시간 (주 40시간 기준)

연차수당 = 미사용 연차일수 × 1일 통상임금

총 수령액 = 퇴직금 + 연차수당
```
