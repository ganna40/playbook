# mz - 엠지력(MZ力) 측정기

> URL: https://mz.pearsoninsight.com
> GitHub: https://github.com/ganna40/mz

## 개요

| 항목 | 내용 |
|------|------|
| **유형** | 선택형 다차원 테스트 |
| **한줄 설명** | 40문항 5지선다 → 8차원 레이더차트 + 7등급 |
| **타겟** | MZ세대 (카톡 바이럴) |
| **테마** | 라이트 (파란 악센트) |
| **폰트** | Pretendard |

## 모듈 조합

```
BASE + DATA + SCREEN + QUIZ + GRADE + RADAR + SHARE + GA + STYLE-LIGHT
```

## 특수 기능

| 기능 | 설명 |
|------|------|
| 5지선다 | "전혀 아니다 ~ 매우 그렇다" 스케일 |
| 8차원 분석 | 카테고리별 점수 → 레이더 차트 |
| 레이더 차트 | Canvas로 시각화 |
| 7등급 | SSS ~ F 등급 |
| 역배점 | 일부 질문은 반전 계산 |

## AI에게 비슷한 거 만들게 하려면

```
playbook의 mz 레퍼런스를 보고
"리더십 유형 검사"를 만들어줘.
[QUIZ] 선택형 + [GRADE] + [RADAR] + [SHARE] + [STYLE-LIGHT]
- 30문항, 5지선다 (전혀~매우)
- 6차원: 소통, 결단, 공감, 전략, 실행, 혁신
- 레이더 차트로 시각화
- 등급: S/A/B/C/D
```
