# amlife - 엠생력 측정기

> URL: https://amlife.pearsoninsight.com
> GitHub: https://github.com/ganna40/amlife

## 개요

| 항목 | 내용 |
|------|------|
| **유형** | 선택형 다차원 테스트 |
| **한줄 설명** | 60문항 → 8차원 + 6등급 "삶의 낭비 측정" |
| **타겟** | 일반 성인 (자기성찰 바이럴) |
| **테마** | 라이트 (파란 악센트) |
| **폰트** | 시스템 폰트 |

## 모듈 조합

```
BASE + DATA + SCREEN + QUIZ + GRADE + RADAR + SHARE + GA + STYLE-LIGHT
```

## mz와의 차이점

| 항목 | mz | amlife |
|------|-----|--------|
| 문항 수 | 40 | 60 |
| 등급 수 | 7 (SSS~F) | 6 (S~F) |
| 톤앤매너 | 트렌디/밝음 | 약간 진지/직설 |
| 데이터 | 인라인 | data.json 분리 |

## AI에게 비슷한 거 만들게 하려면

```
playbook의 amlife 레퍼런스를 보고
"자기관리력 측정기"를 만들어줘.
[QUIZ] 선택형 + [GRADE] + [RADAR] + [SHARE] + [STYLE-LIGHT]
- 50문항
- amlife와 동일한 구조
- data.json 분리
```
