# tok-wrapped - 카카오톡 대화 Wrapped 분석기

> URL: (로컬 / GitHub Pages 배포 예정)
> 로컬: C:\Users\ganna\Downloads\tok-wrapped

## 개요

| 항목 | 내용 |
|------|------|
| **유형** | 파일 분석기 (Wrapped 스타일) |
| **한줄 설명** | 카카오톡 .txt 내보내기 파일을 업로드하면 대화 스타일을 분석하고 Spotify Wrapped처럼 카드 시퀀스로 보여주는 바이럴 웹앱 |
| **타겟** | 카카오톡 사용자 (한국 인구 93%, ~4,700만 명) |
| **테마** | 다크 (카드별 그라데이션) |
| **폰트** | Pretendard Variable (CDN) |

## 모듈 조합

```
BASE + SCREEN + GRADE + REVEAL + SHARE + CALC(숫자애니) + STYLE-DARK + WebWorker
```

## 특수 기능

| 기능 | 설명 |
|------|------|
| **카카오톡 .txt 파싱** | 모바일+PC 내보내기 포맷 정규식 파싱, 멀티라인 처리, UTF-8 BOM 제거, 오전/오후→24시 변환 |
| **Web Worker 파싱** | 대용량 파일 메인 스레드 블로킹 방지, 프로그레스 보고 |
| **16캐릭터 룰기반 판정** | 통계 기반 스코어링으로 AI 없이 캐릭터 유형 결정 |
| **8장 Wrapped 카드** | 전체화면 카드별 그라데이션, 탭 전환, 숫자 카운트업 애니메이션 |
| **컨페티 + slamIn** | 캐릭터 결과 카드에서 극적 공개 연출 |
| **100% 정적** | 백엔드 없음, 서버 비용 ₩0, GitHub Pages 배포 가능 |
| **100% 브라우저 로컬** | 대화 내용이 서버에 전송되지 않음 |
| **참가자 자동 감지** | 1:1은 헤더에서 자동 판별, 그룹은 선택 UI |
| **이미지 저장** | html2canvas로 결과 카드 PNG 저장 |

## 핵심 기술

| 기술 | 용도 |
|------|------|
| Web Worker | 대용량 .txt 파싱 (메인 스레드 보호) |
| 정규식 파서 | 카카오톡 모바일/PC 내보내기 포맷 파싱 |
| 룰 기반 스코어링 | 16개 캐릭터 유형 판정 (AI 없이) |
| html2canvas | 결과 카드 스크린샷 저장 |
| Pretendard Variable | 한글 웹폰트 (CDN) |

## 파일 구조

## 카카오톡 파싱 정규식

```javascript
// 모바일: 2024년 1월 5일 오후 12:15, 박영희 : 메시지
const RE_MOBILE = /^(\d{4})년\s*(\d{1,2})월\s*(\d{1,2})일\s+(오전|오후)\s+(\d{1,2}):(\d{2}),\s*(.+?)\s*:\s*(.+)/;

// PC: [박영희] [오후 2:30] 메시지
const RE_PC = /^\[(.+?)\]\s*\[(오전|오후)\s*(\d{1,2}):(\d{2})\]\s*(.+)/;
```

## 16 캐릭터 유형 (룰 기반 판정)

| # | 캐릭터 | 판정 기준 |
|---|--------|-----------|
| 1 | 리액션 장인 🎭 | ㅋㅋ/메시지 > 25%, 이모지 > 20% |
| 2 | 새벽 감성러 🌙 | 0~4시 비율 > 20% |
| 3 | 번개 답장러 ⚡ | 대화 시작 40~60% + 일평균 50+ |
| 4 | 읽씹 예술가 🐢 | 대화 시작 < 30% |
| 5 | 장문 소설가 📖 | 평균 메시지 > 25자 |
| 6 | 한글자 닌자 🥷 | 한글자 비율 > 30%, 평균 < 5자 |
| 7 | 질문 폭격기 ❓ | 질문 비율 > 20% |
| 8 | 링크 전도사 🔗 | 링크/메시지 > 8% |
| 9 | 이모지 화가 🎨 | 이모지/메시지 > 35% |
| 10 | 대화 스타터 👋 | 먼저 말 건 비율 > 65% |
| 11 | 텀 두는 사람 🏝️ | 활동일/전체일 < 40% |
| 12 | 꾸준한 연락러 📅 | 연속일 > 30일, 활동 비율 > 80% |
| 13 | 사진 폭격기 📸 | 미디어/메시지 > 15% |
| 14 | 감성 리액터 🥲 | ㅠㅠ/메시지 > 10% |
| 15 | 아침형 인간 🌅 | 5~12시 비율 > 30% |
| 16 | 톡 홀릭 🔥 | 일평균 100+ 메시지 |

## 8장 Wrapped 카드 시퀀스

| 카드 | 내용 | 배경 그라데이션 |
|------|------|----------------|
| 1 | 총 메시지 + 기간 + 일평균 | 딥 네이비 |
| 2 | 시간대 히트맵 + 유형(올빼미/아침형) | 남보라 |
| 3 | TOP 5 단어 랭킹 | 핫 핑크 |
| 4 | ㅋㅋㅋ 횟수 + 평균 길이 | 골드→오렌지 |
| 5 | 먼저 말 건 비율 + 대화 주도권 | 그린→민트 |
| 6 | TOP 이모지 + 사용 횟수 | 핑크→퍼플 |
| 7 | 캐릭터 유형 (클라이맥스, 컨페티) | 퍼플 그라데이션 |
| 8 | 공유 (이미지 저장 / 링크 복사) | 핑크 그라데이션 |
