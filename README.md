# Playbook

> 개발 레시피북 + 모듈 카탈로그.
> 로봇 조립하듯이 부품을 끼우면 앱이 완성됨.

## 빠른 시작

### 1. 만들고 싶은 앱 타입 고르기 → [조립 가이드](builder.md)

```
앱 = 뼈대 + 두뇌 + 얼굴 + 입출력 + 기억장치
```

| 타입 | 뭘 만드는 건지 | 대표 작품 |
|------|---------------|-----------|
| 바이럴 테스트 | 공유하는 성격/능력 테스트 | pong, mz, amlife |
| 계산기 | 입력 → 결과 도구 | salary |
| 추천기 | 질문 → 검색 → 추천 | food |
| AI 챗봇 | AI 대화 서비스 | tarot, psycho-bot |
| 알림 봇 | 자동 알림/경고 | telbot |
| 업무 도구 | 팀 협업 대시보드 | collab-tool |

### 2. AI에게 던지기

```
playbook의 조립 가이드를 보고
"___" 앱을 만들어줘.

타입: [바이럴 테스트 / 계산기 / 추천기 / AI 챗봇 / 알림 봇 / 업무 도구]
부품: [필요한 모듈 나열]
참고: [가장 비슷한 레퍼런스]
테마: [다크 / 라이트]
```
