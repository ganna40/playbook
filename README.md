# Playbook

> 개발 레시피북 + 모듈 카탈로그.
> AI한테 이 문서 던지면 바로 구현 가능.

## 사용법

### 새로 만들 때 (AI에게)
```
playbook의 [카탈로그]를 참고해서
[QUIZ] + [KAKAO] + [OG] + [GRADE] + [STYLE-DARK] 모듈을 조합해서
"MBTI 테스트" 앱을 만들어줘.
데이터는 data.json으로 분리하고, 배포는 [DEPLOY] 파이프라인 따라서.
```

### 기존 거랑 비슷하게 만들 때
```
playbook의 salary 레퍼런스를 보고
같은 구조로 "2026 연봉 계산기"를 만들어줘.
[CALC] + [GRADE] + [KAKAO] 모듈 조합.
```

## 구조

| 폴더 | 용도 |
|------|------|
| `catalog/` | **모듈 카탈로그** - 조합 가능한 빌딩블록 |
| `references/` | **레퍼런스** - 실제 만든 앱 분석 |
| `recipes/` | **레시피** - 기술별 How-to |
| `gotchas/` | **삽질 방지** - 실수 모음 |
| `pipeline/` | **파이프라인** - 빌드/배포/운영 |
