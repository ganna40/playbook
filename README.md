# Playbook

> 개발하면서 배운 것들을 정리하는 개인 레시피북.
> AI한테 이 문서 던지면 바로 구현 가능.

## 사용법

### 직접 볼 때
사이드바에서 필요한 레시피를 클릭하세요.

### AI에게 줄 때
```
이 파일을 보고 동일한 구조로 만들어줘:
[레시피 파일 내용 복붙 또는 URL]
```

## 구조

| 폴더 | 용도 |
|------|------|
| `recipes/` | "이걸 만들고 싶을 때" 따라하는 레시피 |
| `gotchas/` | "이거 모르면 삽질함" 주의사항 모음 |
| `templates/` | 복붙 가능한 보일러플레이트 코드 |
